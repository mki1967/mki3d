How to install/use MKI3D RAPID MODELLER

REMARK: Mki3d is now served as Progressive Web App. It should be made usable off-line by most of the modern browsers.

You have the following options of using MKI3D RAPID MODELLER:

Use off-line:
Either
  • Download .zip and unpack, or
  • Download .tar.gz and unpack, or
  • clone the git repository with a command: git clone https://github.com/mki1967/mki3d.git
    (This lets you get updates with simple git pull command in the repo directory.)
The application works in the web-browser (Chrome, Firefox, Opera, ...). To start open in your browser the file file:///path_to_your_unpacked_or_cloned_directory/mki3d_www/index.html
Use on-line:
Open in your web browser the link: http://mki1967.github.io/mki3d/mki3d_www/index.html
Packaged Chrome application:
Use this link for installation of mki3d from Chrome Web Store.
Note that Google decided to start removing support for packaged and hosted apps from Chrome on Windows, Mac, and Linux. Chrome apps will remain supported and maintained on Chrome OS for the foreseeable future. (Read this Chromium blog post.)

Remark: Packaged Chrome application provides file-chooser for saving/loading data. In the remaining cases you have to save/load data through other means. Read How to Use Input and Output through Text-areas.
