Help screen:

Key presses on the canvas:

'H'
- This help screen
'Q'
- Main menu
Arrow Keys, 'F', 'B','V'
- Depending on the MODE, either: move cursor, selected endpoints, or rotate the screen
'A'
- Switch action MODE
Shift
- Switch action MODE to:
  • CURSOR moving - when pressed
  • ROTATION - when released
Space
- Align view rotation to right angles

General description

The model consists of line segments and triangles. Each segment is defined by a pair of points. Each triangle is defined by a triple of points. Each point has position in (x,y,z) coordinates and a color (r,g,b). Each point belongs to a single set. The sets are indexed by natural numbers.

You can find more info on https://github.com/mki1967
